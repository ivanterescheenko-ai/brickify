## Решения
- **Nexar вместо Octopart**: Octopart REST API v4 deprecated (август 2026). Nexar GraphQL — тот же владелец, новый API
- **jlcsearch вместо LCSC reverse-engineered**: бесплатный, без ключа, без лимитов, open source (tscircuit)
- **Mouser а не DigiKey**: DigiKey требует OAuth authorization code flow (сложно для self-hosted). Mouser — простой API key
- **Amazon PA-API отброшен**: deprecating апрель 2026, требует affiliate sales. Используем Tavily site-scoped
- **SSE streaming вместо polling**: real-time progress по этапам pipeline
- **Промпты на английском**: LLM лучше работают на EN, UI тоже на EN для международной аудитории
- **category field в decomposer**: router использует для выбора оптимального search source
