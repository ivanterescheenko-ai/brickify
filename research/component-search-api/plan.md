# Research Plan: API для поиска электронных компонентов и hardware
Дата: 2026-04-13
Domain: tech
Контекст запроса: Пользователь строит Brickify — open source tool с multi-source component search. Нужно полное покрытие доступных API (электроника, механика, крепёж) с лимитами, эндпоинтами и форматами ответов для интеграции.

## Research Questions
1. [web] Какие API предоставляют Octopart, DigiKey, Mouser, Farnell/Element14, Arrow для поиска электронных компонентов? Эндпоинты, аутентификация, бесплатные лимиты, формат ответа.
2. [web] Какие API есть у LCSC, SZLCSC и JLCPCB для поиска компонентов? Документация, ограничения, reverse-engineered endpoints.
3. [web] Amazon Product Advertising API и Google Shopping API — как использовать для поиска hardware компонентов? Условия доступа, лимиты, стоимость.
4. [web] AliExpress API / affiliate programs — есть ли способы программно искать товары и получать цены? Открытые API, партнёрские программы.
5. [discussion] Какие менее известные API и сервисы используют разработчики для поиска электронных компонентов? FindChips, Nexar, SnapEDA, ComponentSearchEngine.
6. [web] Какие open source проекты реализуют поиск компонентов через несколько API? KiCad, IntelligentElectronicComponentSearch, partfinder.
7. [news] Какие новые API и сервисы для поиска компонентов появились в 2025-2026? Тренды, новые игроки.
